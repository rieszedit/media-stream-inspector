Privacy Policy — Media Stream Inspector

Last updated: February 11, 2026

Overview

Media Stream Inspector ("the Extension") is a browser extension that detects and downloads HTML5 media streams (HLS/MP4) from web pages. This privacy policy explains how the Extension handles user data.

Data Collection

The Extension does not collect, store, transmit, or share any personal data. Specifically:

  • No browsing history is collected or transmitted
  • No user credentials or form data are accessed
  • No analytics or tracking tools are used
  • No data is sent to external servers
  • No cookies are read or modified

Local Data Storage

The Extension uses chrome.storage.local solely to store download history (file names and timestamps) on the user's local device. This data never leaves the user's browser and can be cleared at any time by removing the Extension.

Permissions

The Extension requests the following permissions for the stated purposes only:

  • Host permissions (<all_urls>) — Required to detect media streams on any web page the user visits
  • webRequest — Required to monitor network requests for HLS (.m3u8) and MP4 media files
  • webNavigation — Required to detect page navigation and reset media detection state
  • declarativeNetRequest — Required to set appropriate request headers for media downloads
  • downloads — Required to save downloaded media files to the user's device
  • offscreen — Required to process HLS stream segments in the background
  • notifications — Required to notify the user when downloads complete
  • storage — Required to store download history locally
  • alarms — Required to keep the service worker active during downloads
  • activeTab — Required to access the current tab for media detection

Third-Party Services

The Extension does not use any third-party services, APIs, or SDKs.
